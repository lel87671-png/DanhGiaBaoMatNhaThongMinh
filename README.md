# Đánh giá bảo mật mô hình nhà thông minh

## Thông tin sinh viên

- Họ và tên: Lê Quang Long
- MSSV: 231A011163
- Học phần: Bảo mật IoT (INT4410)

## Mô tả đề tài

Đề tài tập trung đánh giá các rủi ro an toàn thông tin trong mô hình nhà thông minh, phân tích các lỗ hổng bảo mật phổ biến và đề xuất các giải pháp nhằm nâng cao mức độ an toàn của hệ thống.
